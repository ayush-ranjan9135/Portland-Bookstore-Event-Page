A beautiful, responsive, and dynamic landing page built for an intimate literary event hosted by Portland's independent bookshop, Folio & Fox Books.

## 📖 Problem Statement
Local independent bookshops often struggle to create a digital presence that matches the intimacy, warmth, and intellectual charm of their physical spaces. When hosting special events like author readings and book signings, standard event platforms feel too clinical and transactional. There is a need for a digital landing page that captures the atmosphere of a cozy bookstore while effectively conveying event details and driving ticket reservations.

## 💡 Solution
We built a visually captivating, atmospheric landing page specifically designed for literary events. By using a warm color palette (deep reds, warm creams, and near-blacks), elegant typography (Cormorant Garamond and DM Sans), and subtle micro-animations, the page feels like an extension of the bookstore itself. It provides all necessary event details—author bios, event schedule, and location—in an engaging, easy-to-navigate format that encourages attendees to reserve their seats.

## 🔄 Project Flow
1. **Hero Section:** Greets users with an atmospheric background, dynamic typographic animations, and clear event metadata (Date, Time, Venue).
2. **The Evening (About):** Sets the tone with a quote and outlines the key features of the event (Live Readings, Open Q&A, Book Signing).
3. **Featured Authors:** Introduces the four guest authors using a responsive CSS grid and engaging hover effects.
4. **Event Details (Ticket Stub):** Displays actionable event information modeled after a classic ticket stub, leading directly to the CTA to reserve a seat.
5. **Location & Footer:** Provides practical visitor information, an embedded map aesthetic, and links to explore further.

## 🏗️ Architecture
The project follows a clean, modular frontend architecture:
- **`prodesk.html`**: The semantic HTML5 backbone of the page, defining the structure and accessibility markers.
- **`styles.css`**: The design system, including CSS variables (custom properties) for theming, flexbox/grid layouts, responsive media queries, and keyframe animations.
- **`script.js`**: Lightweight vanilla JavaScript handling interactive elements like the scroll-reveal effects, dynamic transparent-to-solid navigation bar, and the floating literary glyphs canvas animation in the hero section.

## 💻 Tech Stack
* **Markup:** HTML5 (Semantic & Accessible)
* **Styling:** Vanilla CSS3 (Custom Properties, Flexbox, CSS Grid, Media Queries, Keyframes)
* **Logic & Interactivity:** Vanilla JavaScript (ES6, IntersectionObserver API, HTML5 Canvas)
* **Typography:** Google Fonts (`Cormorant Garamond`, `DM Sans`)
